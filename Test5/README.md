# Kiwi.com Landing Page

A landing page inspired by Kiwi.com, built with React and Kiwi Orbit components.

## Features

- Flight search form with return/one-way options
- Popular destinations section
- Mobile app promotion
- Popular flights grid
- Explore sections (cities, countries, regions)
- Responsive navigation and footer

## Getting Started

### Installation

Install the dependencies:

```bash
npm install
```

### Development

Start the development server:

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

Build for production:

```bash
npm run build
```

### Preview

Preview the production build:

```bash
npm run preview
```

## Tech Stack

- **React** - UI library
- **Vite** - Build tool
- **Kiwi Orbit Components** - Design system components
- **Styled Components** - CSS-in-JS styling

## Components

- `Navigation` - Top navigation bar
- `Hero` - Main hero section with flight search
- `PopularDestinations` - Grid of popular destination cards
- `MobileApp` - Mobile app promotion section
- `PopularFlights` - Grid of popular flight routes
- `ExploreSection` - Explore links for cities, countries, and regions
- `Footer` - Site footer with links

## Design

The design follows Kiwi.com's visual style with:
- Kiwi green color scheme (#00a991)
- Clean, modern UI
- Responsive grid layouts
- Smooth hover animations
- Card-based components
